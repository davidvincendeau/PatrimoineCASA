Patrimoine-CASA
========================

Utilise:
- Ionic Framework 
- [Leaflet.js](http://leafletjs.com).

# Installation
- Installer Ionic CLI

```sh
ionic start Patrimoine-CASA https://github.com/opencasa/Patrimoine-CASA
cd Patrimoine-CASA
cordova plugin add org.apache.cordova.geolocation
ionic platform add ios
ionic platform add android
ionic serve
```
